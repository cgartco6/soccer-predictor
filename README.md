# Soccer Prediction System

A comprehensive AI-powered system for predicting soccer match results (win/draw) and Both Teams to Score (BTTS) outcomes.

## Features

- **Real-time Data Collection**: Scrapes match data from Hollywoodbets and Betway
- **Advanced Feature Engineering**: 50+ features including player conditions, team performance, weather, and more
- **Machine Learning Models**: Ensemble models for result and BTTS predictions
- **Continuous Learning**: Models improve automatically with new data
- **Value Bet Detection**: Identifies profitable betting opportunities
- **REST API**: Easy integration with other systems
